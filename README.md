# Ponto Eletrônico Web

Uma aplicação web simples para registrar horas trabalhadas, com interface amigável e armazenamento local.

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Use os botões para registrar seus pontos:
   - Primeira Entrada
   - Primeira Saída
   - Segunda Entrada
   - Segunda Saída
   - Outro (para registros personalizados)
3. Visualize seus registros do dia na seção "Registros de Hoje"
4. Exporte os registros para um arquivo de texto clicando em "Exportar para TXT"

## Funcionalidades

- Registro de pontos com data e hora
- Interface intuitiva e responsiva
- Armazenamento local no navegador (não perde os dados ao atualizar a página)
- Exportação dos registros para arquivo de texto
- Funciona offline após carregado

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, Safari)
- Nenhuma instalação adicional necessária

## Observações

- Os dados são armazenados no localStorage do navegador
- Os registros são organizados por data
- A exportação gera um arquivo de texto com todos os registros do dia
